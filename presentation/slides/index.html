The Foundation

LLM Architecture

From words → vectors → attention → prediction

What Is an LLM, Really?

A next-token prediction machine

Every LLM capability emerges from this single objective: minimize prediction error on the next token

Embeddings — Words as Vectors in Space

Each word becomes a point in a high-dimensional space where geometry encodes meaning

Mikolov et al., 2013, "Efficient Estimation of Word Representations in Vector Space," arXiv:1301.3781 · Visual inspired by 3Blue1Brown "But what is a GPT?" (2024)

Directions Encode Meaning

In embedding space, consistent directions capture consistent relationships

These relationships are not programmed — they emerge from seeing words in similar contexts during training

So We Have Vectors... Now What?

The problem: Every word starts as an isolated vector. "Bank" has the same embedding whether it means river bank or financial bank.

We need context. The word's meaning depends on the words around it.

Attention solves this: it lets every token look at every other token and decide "who should I listen to?"

Embedding → Attention (contextualize) → Rich, context-aware representation

This is the core insight of Transformers: embeddings give you meaning; attention gives you context

MLP Layers — Where Facts Live

Research evidence:

  • Specific neurons encode specific facts
  • Suppressing neurons removes the fact
  • MLP acts as a key-value memory
  • ~66% of GPT-3's parameters are in MLPs
Attention = wiring/routing
MLP = the actual content

Dai et al., 2022, "Knowledge Neurons," ACL, arXiv:2104.08696 · Geva et al., 2021, "FFN as Key-Value Memories," EMNLP, arXiv:2012.14913

Why Transformers Won

✅ The Wins

  • Fully parallel — 100× faster training than RNNs
  • O(1) path between any two positions
  • BLEU 28.4 EN→DE (new SOTA at fraction of cost)
  • Set the stage for everything after 2017

⚠️ But Not the End

  • O(n²) attention — 128K tokens = 16.4B entries
  • Autoregressive ≠ planning or understanding
  • No world model — can't verify its own claims
  • These motivate the next frontier...
Swaroop Mishra's reasoning work (Step-Back, Self-Discover) provides workarounds for the planning limitation — but architecturally, the limitation remains. Can we build systems that plan natively?

Vaswani et al., 2017 · Zheng, Mishra et al., 2023, "Step-Back Prompting," ICLR 2024 · Zhou, Mishra et al., 2024, "Self-Discover," Google DeepMind

The Frontier — Beyond Token Prediction

JEPA · Joint Embedding Predictive Architecture

  • I-JEPA (Meta, 2023): Predicts masked image patches in representation space
  • V-JEPA (Meta, 2024): Learns video dynamics without generating pixels
  • Not a replacement today — a research frontier addressing fundamental limitations
