# Project: Generative AI & LLMs — Research-Grounded Presentation

## Purpose
Build a comprehensive, research-backed presentation on LLM architectures, fine-tuning, LoRA, and API integrations. The presentation must:
- Teach concepts step-by-step with visual aids
- Ground everything in original research papers and verified sources
- Separate hype from substance using first-principles reasoning
- Inspire the audience to innovate by understanding *why* things work

## Core Principles
1. **Primary sources first** — Always reference the original paper where a concept was introduced
2. **Verified claims only** — Every statistic, benchmark, or comparison must link to a verifiable source
3. **Visual learning** — Diagrams, architecture visuals, and data charts over walls of text
4. **Hype detection** — Clearly distinguish what's proven vs. what's marketing
5. **Builder mindset** — Teach the methodology so people can create, not just consume

## Target Audience
- Technical professionals who want to understand LLMs deeply
- People who want to build on top of or innovate within this space
- Those tired of social media hype and want grounded understanding

## Presentation Flow
1. Foundations — Transformer architecture (Attention Is All You Need)
2. Scaling — How LLMs evolved from GPT-1 to modern models
3. Training — Pre-training, instruction tuning, RLHF
4. Fine-tuning — Full fine-tuning vs parameter-efficient methods
5. LoRA & variants — The paper, math, practical usage
6. API Integrations — Building with LLM APIs, RAG, agents
7. What's hype vs. real — Critical analysis with evidence
8. Latest developments — State of the field in 2024-2026
9. Call to action — How to innovate using these foundations

## File Organization
```
/research-papers/       — Downloaded papers and summaries
/research-papers/summaries/ — One-page summaries per paper
/presentation/          — Final presentation content
/presentation/visuals/  — Diagrams, charts, images (with attribution)
/notes/                 — Working notes, filtered information
/references/            — Blog posts, articles, verified sources
```
